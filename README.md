# nir4ist
Шаблон отчета по НИР для студентов направления "Инф.системы и технологии" ИАТЭ НИЯУ МИФИ

Проект перенесен с GitHub, там больше не поддерживается

Основные файлы (выберите подходящий):

NIR_otchet.tex - шаблон для отчета по НИР

BAK_vkr.tex - шаблон для бакалаврской работы

MAG_disser.tex - шаблон для магистерской диссертации

MSKMAG_disser.tex - шаблон, использовавшийся для защит в 2018 г.

Все шаблоны используют вспомогательные и включаемые файлы:

rbiblio.bib - пример библиографичской базы данных BibTeX (в примерах используется biblatex и biber - укажите biber в настройках TeXstudio!)

intro.tex - пример введения

part1.tex, part2.tex - основные разделы документа, при необходимости их можно добавлять

Примеры использования библиографии, вставки рисунков и таблиц приведены в тексте.

Важные изменения от 2024 г.
ГОСТ 7.32-2017 - нумерация страниц не проставляется только на титульной странице, поэтому включена нумерация всех страниц, начиная с номера 2 на реферате.
Добавлено упрощённое окружение MyList для вставки исходных кодов, название листинга и его метка для ссылок указываются как параметры. Непосредственно код вставляется либо с помощью окружений MyCode, либо MyCodes

```
\begin{MyList}{Название листинга}{Метка листинга для ссылки в тексте, напр., lst:10}
\begin{MyCode}
код вставляется сюда
\end{MyCode}
\end{MyList}
```

Для сборки примера (например, NIR_otchet.tex) в консоли следует выполнить команды

```
pdflatex NIR_otchet.tex
biber NIR_otchet
pdflatex NIR_otchet.tex
```
